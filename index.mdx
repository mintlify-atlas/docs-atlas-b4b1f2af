## What's Included

This starter kit provides everything you need to build production-ready documentation:

- **Pre-configured structure** - Navigation, settings, and pages ready to customize
- **Component examples** - Cards, accordions, code blocks, tabs, and more
- **AI tool integration** - Skills and configurations for Cursor, Claude Code, and Windsurf
- **API reference template** - OpenAPI integration with example endpoints
- **Best practices** - Writing guidelines and content organization patterns
- **Local development** - CLI tools for instant preview and validation

<Note>
  Start by customizing the `docs.json` file to match your project name and brand colors, then explore the example pages to see Mintlify's features in action.
</Note>
