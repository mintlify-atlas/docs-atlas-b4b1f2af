## Inline code

To denote a `word` or `phrase` as code, enclose it in backticks (`).

```md

```

## Code blocks

Use [fenced code blocks](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) by enclosing code in three backticks and follow the leading ticks with the programming language of your snippet to get syntax highlighting.

### Basic syntax

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet('World');
```

````md
```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet('World');
```
````

### With filename

Optionally, you can write the name of your code file after the programming language:

```java HelloWorld.java

        System.out.println("Hello, World!");
    }
}
```

## Line highlighting

Highlight specific lines in your code blocks to draw attention to important parts:

```javascript {2,5-7}
function calculateTotal(items) {
  let total = 0; // This line is highlighted
  
  items.forEach(item => {
    total += item.price; // These lines
    total += item.tax;   // are also
    total -= item.discount; // highlighted
  });
  
  return total;
}
```

## CodeGroup

Use CodeGroup to display multiple code examples in tabs, perfect for showing the same functionality in different languages:

<CodeGroup>

```javascript JavaScript
const response = await fetch('https://api.example.com/users', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    name: 'Alice',
    email: 'alice@example.com'
  })
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.post(
    'https://api.example.com/users',
    json={
        'name': 'Alice',
        'email': 'alice@example.com'
    }
)

data = response.json()
print(data)
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "net/http"
)

func main() {
    user := map[string]string{
        "name":  "Alice",
        "email": "alice@example.com",
    }
    
    jsonData, _ := json.Marshal(user)
    resp, _ := http.Post(
        "https://api.example.com/users",
        "application/json",
        bytes.NewBuffer(jsonData),
    )
    defer resp.Body.Close()
}
```

```bash cURL
curl -X POST https://api.example.com/users \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Alice",
    "email": "alice@example.com"
  }'
```

</CodeGroup>

## Best practices

<AccordionGroup>
  <Accordion title="Always specify the language" icon="language">
    Always include the language identifier for proper syntax highlighting:
    
    ✅ Good: ` ```javascript`
    
    ❌ Bad: ` ``` `
  </Accordion>
  
  <Accordion title="Keep examples concise" icon="scissors">
    Show only the relevant code. Remove boilerplate and focus on what matters:
    
    ```javascript
    // Good: Focused example
    const result = await api.createUser(data);
    
    // Bad: Too much context
    import { api } from './api';
    import { config } from './config';
    import { logger } from './logger';
    // ... many more imports ...
    const result = await api.createUser(data);
    ```
  </Accordion>
  
  <Accordion title="Add comments for clarity" icon="comment">
    Use comments to explain non-obvious code:
    
    ```python
    def calculate_score(value):
        # Normalize the value to 0-100 range
        normalized = (value - min_value) / (max_value - min_value) * 100
        return normalized
    ```
  </Accordion>
  
  <Accordion title="Use CodeGroup for multi-language examples" icon="layer-group">
    When showing the same functionality in different languages, use CodeGroup instead of separate code blocks. This improves readability and lets users quickly switch between languages.
  </Accordion>
</AccordionGroup>
